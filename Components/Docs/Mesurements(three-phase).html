Three-Phase Measurement

The three-phase electrical quantities at a bus from a dynamic simulation can be captured using the measurement block. These signals typically include: voltage magnitude, frequency, real power (P), reactive power (Q), and three-phase voltage and current waveform. All the quantities are captured in per-unit (pu) with the base values provided in the block mask.
The block needs to be connected to a three-phase physical connection point like an electrical node. The in(abc) and out(abc) are the physical input port and output port of the block. The Data port provides the measurements as a bus signal data for post processing, you can store these measurements in a workspace variable using this port.
